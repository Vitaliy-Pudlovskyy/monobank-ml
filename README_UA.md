# 💳 Monobank ML — Аналіз та Передбачення Витрат

Проект машинного навчання на основі особистої банківської виписки Монобанку за 2025 рік. Реалізований з нуля — без ML фреймворків, тільки Python, NumPy і Pandas.

---

## 🎯 Задачі

| Задача | Тип | Файл |
|--------|-----|------|
| Передбачити суму витрат на день | Регресія | `regression.py`, `decision_tree_regression.py` |
| Передбачити категорію витрат на день | Класифікація | `classification.py`, `decision_tree.py`, `knn.py`, `random_forest.py` |
| Порівняти всі моделі | Обидва | `comparison.py` |

---

## 📁 Структура проєкту

```
monobank-ml/
├── data_set
│   └── monobank_clean.csv        # анонімізована виписка
├── regression.py                 # лінійна регресія + нейронна мережа
├── classification.py             # класифікація нейронною мережею + інтерактивний режим
├── decision_tree.py              # дерево рішень (класифікація)
├── decision_tree_regression.py   # дерево рішень (регресія)
├── knn.py                        # K найближчих сусідів
├── random_forest.py              # випадковий ліс
├── comparison.py                 # запускає всі моделі і виводить таблицю порівняння
└── README.md
```

## 🔬 Архітектура

**Лінійна регресія:** `6 → 1`

**Нейронна мережа (класифікація):** `6 → 10 → 6 → 3` з ReLU + Softmax

**Нейронна мережа (регресія):** `6 → 6 → 3 → 1` з ReLU

**Дерево рішень:** максимальна глибина = 3, критерій Джині

**Випадковий ліс:** 6 дерев, беггінг + випадкові підвибірки ознак, голосування більшістю

**KNN:** k=3, манхетенська відстань
